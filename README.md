# 🚀 Student Habit Analyzer

A simple but powerful web application that helps students analyze their daily habits and improve productivity.

This project allows users to track important lifestyle factors such as sleep, study time, sport activity, mood, and phone usage. Based on these inputs, the system calculates a Productivity Score and provides recommendations for improvement.

---

## ✨ Features

- 📊 Habit Tracking  
  Track daily habits including sleep, study, sport, mood, and phone usage.

- 🧠 Productivity Score  
  The app calculates a productivity score based on your daily routine.

- 📈 Dashboard with Charts  
  Visualize your habits using graphs and statistics.

- 🗂 Habit History  
  All past records are stored so you can monitor your progress.

- 🤖 Smart Advice System  
  The app gives recommendations on how to improve productivity.

---

## 🛠 Tech Stack

- Python
- Flask
- HTML
- Chart.js
- JSON (data storage)

---

## 📷 How It Works

1. The user enters daily habit data:
   - Sleep hours
   - Study hours
   - Sport activity
   - Mood level
   - Phone usage

2. The system analyzes the data.

3. A Productivity Score is generated.

4. The dashboard displays charts and habit history.

---

## 🎯 Purpose of the Project

This project was created to explore how simple data analysis can help students better understand their daily habits and optimize their productivity.

It is also a beginner-friendly project demonstrating how to build a full-stack web application using Python and Flask.
